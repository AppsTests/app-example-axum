# Axum

Пример приложения Axum, которое можно развернуть в Timeweb Cloud Apps без настройки.

🎉 [Демо](https://twc-app-example-axum.twc1.net/)

🚀 [Создать свой Apps](https://timeweb.cloud/my/apps/create)

📚 [Документация Timeweb Cloud Apps](https://timeweb.cloud/docs/apps)

## <a name="dev"></a>Локальный запуск проекта
```bash
# установка зависимостей
cargo fetch

# запуск в дев режиме
cargo run

# сборка для продакшн
cargo run --release
```
